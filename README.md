Herramienta de inteligencia artificial tipo chatbot, basada en DescartesJS y Pollinations. La interfaz permite a los usuarios introducir un prompt en un cuadro de texto y obtener respuestas generadas automáticamente. Incorpora hasta diez modelos LLM, así como la integración de imágenes procedentes de Lexica y la posibilidad de convertir el texto generado en formato Markdown a documentos Word.
